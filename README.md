## 初始化项目

机器环境预装配置：Node Git Npm

下载 pnpm

```zsh
npm i pnpm -g
// Mac环境
pnpm setup && source .zshrc
```

配置 pnpm 镜像源

```shell
pnpm config set registry https://registry.npmmirror.com/
```

安装项目依赖

```shell
pnpm i
```

启动项目

```shell
pnpm run dev
```
